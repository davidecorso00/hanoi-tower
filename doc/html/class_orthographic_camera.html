Detailed Description

Camera implementing an orthographic projection.

This class extends the Camera class by providing an orthographic projection. It is typically used for:

  • 2D rendering
  • minimaps
  • UI overlays
  • technical and CAD-style views

Unlike perspective projection, object sizes do not change with distance from the camera.

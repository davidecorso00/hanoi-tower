Detailed Description

OVO file parser and scene loader.

OvoReader is responsible for reading binary .OVO files and converting them into a scene graph composed of Nodes, Meshes, Materials, Textures and Lights.

The parsing is performed recursively, following the hierarchy encoded in the file.

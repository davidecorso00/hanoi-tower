Detailed Description

Represents a 3D mesh in the scene.

A mesh is a geometric object composed of vertices, normals, texture coordinates and faces.

◆ render()

void Mesh::render ( )
overridevirtual

Renders the mesh.

Renders the mesh using OpenGL pipeline.

Applies the associated material and draws the mesh using OpenGL primitives.

If a material is associated, it is applied before rendering. Otherwise, a default rendering state is used. STEPS

  1. Apply the material if it's not nullptr
  2. Apply the geometry
  3. If no material iis applied, restore the OpenGL state

◆ renderShadowGeometry()

void Mesh::renderShadowGeometry ( ) const

Renders geometry only (no material, no lighting state changes).

Draws only the mesh triangles without applying a material.

Used for auxiliary passes such as planar shadows where only the raw triangles are required.

Lighting and texturing should be configured by the caller before invoking this method.
